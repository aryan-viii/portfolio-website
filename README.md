# Personal Portfolio Website (Static Version)

This is the static version of a personal portfolio website created for academic submission.
It focuses on frontend design, responsiveness, and client-side interactivity.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Google Fonts

## Features
- Responsive portfolio website
- Sections: About, Services, Projects, Contact
- Sticky navigation bar
- Clean UI design
- Client-side project management using localStorage
- No backend required
- SEO-friendly structure

## Project Structure

portfolio-static/
├── index.html
├── css/
├── assets/
├── script.js
├── docs/
└── README.md

## Data Handling
- Project data is stored in browser localStorage
- Allows adding projects dynamically
- Data persists after page refresh

## Hosting

The static version of this portfolio website is hosted using **GitHub Pages** and is publicly accessible at:
