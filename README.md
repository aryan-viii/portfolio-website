# Personal Portfolio Website (Full Stack Version)

This is the full stack version of a personal portfolio website developed as part of an academic assignment.
It demonstrates frontend development along with backend integration and CRUD operations.

## Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Google Fonts

### Backend
- Node.js
- Express.js

### Data Storage
- JSON file (server-side storage)

## Features
- Responsive portfolio layout
- Sections: About, Services, Projects, Contact
- CRUD operations for managing projects
- REST API using Express
- Dynamic rendering of project data
- Image upload functionality
- SEO-friendly HTML structure

http://localhost:3000
